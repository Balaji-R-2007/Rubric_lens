## ðŸ“¦ Installation (for Developers)

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/rubric-lens.git
   cd rubric-lens
   ```

2. Install dependencies:
   - For backend:
     ```bash
     cd backend
     npm install
     ```

   - For frontend:
     ```bash
     cd frontend
     npm install
     ```

3. Create `.env` files in both folders with appropriate API keys and config.

4. Run the app:
   - Start backend:
     ```bash
     npm run dev
     ```

   - Start frontend:
     ```bash
     npm start
     ```
